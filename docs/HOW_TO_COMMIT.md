# How to commit

- Never include AI or agent attribution in a commit: no "Co-Authored-By: Claude ...", no "Generated with Claude Code", no mention of which agent, tool, or model wrote the change. The author is the human owner of the repo.
- Commit messages follow [HOW_TO_WRITE_DOCUMENTS.md](HOW_TO_WRITE_DOCUMENTS.md) where it applies: no em-dashes, sober tone, no inflated language.
- Subject line: imperative and concise, saying what the change does ("add zone engine", "move camera credentials to .env").

More rules will be added here as the project grows.
